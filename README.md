# FamHub — hunck.ai

## First time setup (do once)

1. Install Node.js from https://nodejs.org (LTS version)

2. Open Command Prompt in this folder and run:
   npm install

3. Open the .env file and paste your Google Client Secret:
   GOOGLE_CLIENT_SECRET=your-secret-here

4. In Google Cloud Console, add this redirect URI:
   https://hunck.ai/auth/callback

5. Double-click START.bat to launch FamHub

6. Open https://hunck.ai in your browser

## Running FamHub

Just double-click START.bat anytime.
FamHub will be available at https://hunck.ai for the whole family.

## Auto-start on Windows boot

1. Press Win+R, type: shell:startup, press Enter
2. Copy a shortcut to START.bat into that folder
FamHub will now start automatically whenever the computer boots.

## Family members

Each person visits https://hunck.ai and clicks "Sign in with Google"
No app to install — works on any phone, tablet, or computer.
